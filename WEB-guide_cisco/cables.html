CABLES

Dans Cisco Packet Tracer, la section des câbles permet de connecter les différents périphériques réseau entre eux. Vous pouvez choisir parmi une variété de câbles tels que des câbles Ethernet, des câbles fibre optique, des câbles console, etc.

CABLAGE AUTOMATIQUES

Choisi automatiquement le type de cable qui convient entre les appareils

CONSOLE

Le câble qui te permet de te connecter directement au routeur ou au switch pour le configurer, même si ses interfaces réseau sont pas encore réglées

COPPER STRAIGHT-THROUGHT

Le câble réseau classique, celui qu'on utilise pour relier des appareils différents entre eux

COPPER CROSS-OVER

Le câble copper cross-over, c'est le câble réseau qui sert à relier deux appareils du même type entre eux

FIBER

C'est le cable qu'on utilise pour les liaisons très rapides et sur de longues distances, comparé au cuivre classique

PHONE

C'est tout simplement le câble utilisé pour connecter des téléphones analogiques et centrales téléphoniques

COAXIAL

C'est le câble qu'on utilise pour les réseaux anciens ou spécifiques, par exemple les réseaux TV, câble ou certains LAN historiques

SERIAL DCE

C'est le câble qu'on utilise pour simuler les liaisons série entre routeurs, genre les anciennes connexions WAN

SERIAL DTE

C'est la face « réceptrice » dans une liaison série entre routeurs

OCTAL

C'est un câble un peu spécial utilisé pour connecter plusieurs lignes série à la fois sur certains équipements comme les routeurs modulaires

IOT CUSTOM CABLE

C'est un câble qu'on utilise pour connecter des appareils IoT spécifiques entre eux ou avec des routeurs/switchs, mais de façon personnalisée selon le scénario

USB

C'est le câble qu'on utilise pour connecter certains périphériques aux PC ou aux routeurs, surtout pour des appareils IoT ou modules USB dans les scénarios
